# 云栖小院设计系统

## 定位

云栖小院是面向顾客的昆明精品民宿官网。视觉目标是高级、温暖、克制、真实、有云南地域气息，避免普通酒店模板、蓝紫科技风和卡片堆积感。

## 色彩

- 页面暖白：`#F6F1E8`
- 卡片暖白：`#FCF9F4`
- 陶土品牌色：`#A85432`
- 陶土深色：`#7C3E27`
- 深棕文字：`#35231B`
- 辅助暖灰：`#74675F`
- 鼠尾草绿：`#74806B`
- 浅沙色：`#D8C4AA`
- 线条色：`rgba(93, 65, 48, 0.14)`

## 版心与断点

- 页面最大宽度：`1180px` 到 `1240px`。
- 桌面基准：`1440px`。
- 平板断点：`1024px`。
- 手机断点：`768px`。
- 小屏支持：`360px`、`375px`、`390px`、`430px`。
- 全局左右边距使用 `clamp(18px, 4vw, 64px)`。

## 排版

- 字体栈：系统中文无衬线优先，局部品牌标题可使用 Georgia 与中文系统字体组合。
- Hero 标题：桌面 `clamp(48px, 6vw, 92px)`，手机 `clamp(34px, 10vw, 52px)`。
- 区块标题：桌面 `clamp(34px, 4vw, 58px)`，手机 `clamp(28px, 8vw, 38px)`。
- 正文：`16px` 到 `18px`，行高 `1.75` 到 `1.9`。
- 关键中文标题必须用明确 `span` 控制换行，避免单字孤行。

## 布局节奏

- 大区块上下间距：桌面 `96px` 到 `128px`，手机 `64px` 到 `84px`。
- 卡片内部 padding：`20px`、`24px`、`32px`。
- 卡片圆角：普通 `20px` 到 `28px`，图片卡 `24px`。
- 按钮高度：桌面 `50px` 到 `58px`，手机至少 `48px`。
- 阴影：以暖棕低透明度为主，不使用强烈发光。

## 图片策略

- 首屏使用本地 WebP，桌面和手机分尺寸加载。
- 非首屏图片懒加载。
- 真实房型暂用“氛围展示图”标注，避免把图库图冒充真实客房。
- 昆明与滇池素材优先使用许可清晰的 Wikimedia Commons。
- 生活方式和院落素材使用 Pexels / Unsplash，并在 `docs/asset-sources.md` 记录来源。

## 交互与动效

- 全站使用一个滚动显现系统：轻微上移、透明度变化、一次播放。
- Hover：图片轻微放大，卡片微抬升，按钮颜色略深。
- 首屏保留轻微视差，但移动端降低。
- 支持 `prefers-reduced-motion`。
- 不使用粒子、强光圈、夸张 3D、弹跳或高频动画。

## 组件语言

- 导航：暖白半透明、滚动后轻微压缩和阴影。
- 主 CTA：陶土棕实心按钮。
- 次 CTA：暖白描边按钮。
- 房型：一张主推大卡 + 两张辅助卡，桌面错位，手机纵向。
- 智能管家：以对话窗口为中心，侧边承载路线/房型建议。
- 昆明路线：时间线或地图感布局，强调从民宿出发、推荐时长和适合人群。
- 联系区：浅米色或暖白底，二维码清晰但不过度突出。
